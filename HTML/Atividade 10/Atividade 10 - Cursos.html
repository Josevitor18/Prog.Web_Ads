HTML

O que é?

HTML (acroônimo para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilização para produzir páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Sobre o curso:

Este curso ensina desde os conceitos básicos da linguagem HTML, até a criação de formulários para a Web, ideal para iniciantes, e para quem quer um guia de cunsulta, também possui um Glossário com todas as palavras mais utilizadas na internet.

DreamWeaver

O que é?

O Macromedia DreamWeaver é um software de criação para a web criada pela Macromedia (agora Adoe-be Systems), que está atualmente na versão CS3. Versões iniciais da aplicação serviam como um simples editor HTML WYSIWYG("What You See Is What You Get"), permitindo que os usuários criassem páginas da web sem a necessidade de conhecimento profundo de codificação HTML.

Sobre o curso:

Este curso abrange todas as funcionalidades essenciais do DreamWeaver, desde a criação de páginas da web simples até a construção de sites complexos e dinâmicos. Os participantes aprenderão a utilizar as ferramentas visuais do DreamWeaver para criar layouts, inserir conteúdo multimídia, gerenciar projetos e publicar seus sites na internet. Além disso, o curso também aborda técnicas avançadas, como a integração de scripts e estilos CSS, para criar sites responsivos e modernos.

Flash

O que é?

Flash é uma tecnologia de software desenvolvida pela Macromedia (posteriormente adquirida pela Adobe Systems) para criar animações, jogos, aplicativos interativos e conteúdo multimídia para a web. Anteriormente muito popular, o Flash permitia aos designers criar elementos altamente interativos e dinâmicos para websites.

Sobre o curso:

Este curso abrange os princípios básicos do Flash, desde a criação de animações simples até a construção de aplicativos interativos e jogos para a web. Os participantes aprenderão a utilizar as ferramentas de design e animação do Flash, bem como a linguagem de programação ActionScript, para criar conteúdo multimídia envolvente. O curso também aborda técnicas avançadas, como a integração de áudio e vídeo, para criar experiências de usuário ricas e imersivas.

Fireworks

O que é?

Fireworks é uma tecnologia de software desenvolvida pela Macromedia (posteriormente adquirida pela Adobe Systems) para criação e edição de gráficos vetoriais e bitmap para web e dispositivos móveis. Embora menos conhecido que o Flash, o Fireworks é amplamente utilizado por designers web para criar layouts de websites, botões, ícones e outros elementos gráficos.

Sobre o curso:

Este curso aborda os conceitos fundamentais do Fireworks, desde a criação de gráficos simples até a elaboração de designs complexos para web. Os participantes aprenderão a utilizar as ferramentas de desenho vetorial e edição de bitmap do Fireworks para criar e editar elementos gráficos para websites e aplicativos móveis. O curso também aborda técnicas avançadas, como a criação de wireframes e protótipos de interfaces de usuário, para facilitar o processo de design e desenvolvimento web.
